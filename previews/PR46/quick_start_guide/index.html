Quick Start Guide

You can access example data in the Power Systems Test Data Repository, the data can be downloaded with the PowerSystems.jl submodule UtilsData

using PowerSystems
DATA_DIR = download(PowerSystems.UtilsData.TestData, folder = pwd())

Loading data

Data can be loaded from several file formats and return a summary of the system's components and time-series.

julia> using PowerSystems


julia> system_data = System(joinpath(DATA_DIR, "matpower/RTS_GMLC.m"))
ERROR: SystemError: opening file "../../data/matpower/RTS_GMLC.m": No such file or directory
